Somechain

Somechain (Proof of Work Blockchain), a Learning framework for blockchain programming and development is created at VJTI (It is written from scratch using Python and Flask).

Somechain code repository is on GitHub. After experimenting with Somechain Learning framework (PoW – scalability/validation latency), we created VJTI Blockchain (Proof of Authority – for fast validation).


VJTI Chain

VJTI Blockchain is a campus blockchain where coins can be earned by solving course assignments on time and coins can be consumed at VJTI cafeteria and book store.


Explorer and dashboard to view all transactions in real time.

The code is open sourced for VJTI Chain and the wallet on GitHub
